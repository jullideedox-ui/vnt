# Neutral Contact Form

This is a simple, safe "Contact Us" page that won't trigger browser warnings. 

- Fields: Name, Email, Message
- Submits to tushwire@live.com via FormSubmit
- Clean design, safe for GitHub Pages

## How to host on GitHub Pages:
1. Create a new repository on GitHub.
2. Upload this HTML file (rename to index.html if desired).
3. Go to Settings > Pages, choose Deploy from Branch, set branch to main, folder to /root.
4. Save and your contact form will be live at https://<username>.github.io/<repo-name>/
